# 好友功能模块用户操作手册

## 1. 进入好友系统
1. 登录学生端账号。
2. 在首页仪表盘的"快捷操作"区域，点击"好友系统"卡片；或者在个人中心侧边栏（如果已添加）访问。

## 2. 添加好友
1. 在好友页面右上角点击"添加好友"按钮。
2. 在弹出的对话框中，输入目标用户的用户名或姓名进行搜索。
3. 在搜索结果列表中，点击"添加"按钮发送好友请求。
   - 注意：每天的好友请求发送频率有限制（5次/分钟）。
   - 如果对方设置了"禁止被添加为好友"，则无法发送请求。

## 3. 处理好友请求
1. 当有新的好友请求时，好友页面的"好友请求"标签页会出现红点或数字提示。
2. 点击"好友请求"标签页，查看待处理的请求列表。
3. 点击"接受"按钮同意请求，双方正式成为好友；点击"拒绝"按钮忽略该请求。

## 4. 管理好友
1. 在"好友列表"标签页，可以看到所有已添加的好友。
2. 好友卡片显示好友的头像、姓名和在线状态（在线/离线）。
3. 如需删除好友，点击卡片右侧的"删除"按钮，在弹出的确认框中点击"确定"即可解除好友关系。

## 5. 隐私设置
1. 点击页面右上角的头像，选择"个人中心"。
2. 进入"账号安全"页面。
3. 在"隐私设置"区域，可以通过开关控制"允许被添加为好友"。
   - 开启（默认）：允许其他用户搜索并添加您。
   - 关闭：其他用户无法向您发送好友请求。
